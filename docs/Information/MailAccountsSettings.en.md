# Mail Accounts Settings

Configure email account for users and profiles by overwriting **web.config** SMTP settings. In case you configure mail setting account for an user it'll take it to send mails. In case you configure mail setting account for a profile it'll take it to send mail. If you related both it'll use the user configuration. If there aren't any configuration, the application will take the web.config SMTP settings.

## Create Mail Account

You can manage the mail accounts settings from **Admin Panel Area > Security > Mail Account**.

![](../docs_assets/images/MailAccountsSettings/AdminWorkArea.PNG "Image 1. Admin Work Area")

Complete the mail server and account info to send mails from a specific configuration.

![](../docs_assets/images/MailAccountsSettings/NewMailAccountSetting.PNG "Image 2. New Mail Account Setting")

## Link Mail Account

You can link email account settings to users in **Admin Work Area > Security > Users**.

![](../docs_assets/images/MailAccountsSettings/MailAccountUsers.PNG "Image 3. Mail Account Users")

You can link email accounts settings to profiles in **Admin Work Area > Security > Profiles**.

![](../docs_assets/images/MailAccountsSettings/MailAccountProfiles.PNG "Image 4. Mail Account Profiles")

## Testing Mail Account

You have two different options to test email account settings.

1\. You can test mail configuration from a specific mail account settings or completing the mail account configuration by yourself.

![](../docs_assets/images/MailAccountsSettings/MailAccountSettingProcesses.PNG "Image 5. Mail Account Setting Processes")

2\. You can test a **specific user mail account** configuration by sending a test mail.

![](../docs_assets/images/MailAccountsSettings/MailAccountMailTest.PNG "Image 6. Mail Account Mail Test")

Test any mail configuration from collection process by sending a test mail.

![](../docs_assets/images/MailAccountsSettings/MailAccountsSettingsCollectionProcesses.PNG "Image 7. Mail Accounts Settings Collection Processes")

![](../docs_assets/images/MailAccountsSettings/MailAccountsSettingsMailTest.PNG "Image 8. Mail Accounts Settings Mail Test")

## Change Mail Account Password

You can change the mail account password as Admin for every user.

![](../docs_assets/images/MailAccountsSettings/MailAccountChangePassword.PNG "Image 9. Mail Account Change Password")

Any user who has an email account related can change the password from their control panel.

![](../docs_assets/images/MailAccountsSettings/UserChangeMailPassword.PNG "Image 10. User Change Mail Password")
