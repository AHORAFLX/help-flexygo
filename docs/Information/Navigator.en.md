# Navigator Module

The **navigator** has all necesary logic to enable navigation to different targets and to differrent node types given an initial node Id.

## Navigation mode

![Navigation mode](../docs_assets/images/Navigator/Navigation.png#only-light "Navigation mode"){data-gallery="light"}
![Navigation mode](../docs_assets/images/Navigator/Navigation_dark.png#only-dark "Navigation mode"){data-gallery="dark"}

```html
<flx-nav id="mainNav" mode="nav" initnode="mainBar"></flx-nav>
```
