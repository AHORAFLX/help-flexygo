# Useful tokens

Use this tokens when you're developing your app to obtain some useful information just by adding them to your code.

*   <fh-copy>{{currentDate}}</fh-copy>: Get current date, useful as default value
*   <fh-copy>{{currentTime}}</fh-copy>: Get current time, useful as default value
*   <fh-copy>{{currentDateTime}}</fh-copy>: Get current date and time, useful as default value
*   <fh-copy>{{currentLocation}}</fh-copy>: Get current user location (altitude, longitude), useful as default value
*   <fh-copy>{{currentAltitude}}</fh-copy>: Get current user altitude, useful as default value
*   <fh-copy>{{currentLongitude}}</fh-copy>: Get current user longitude, useful as default value
*   <fh-copy>{{currentLatitude}}</fh-copy>: Get current user latitude, useful as default value
*   <fh-copy>{{newGuid}}</fh-copy>: Generate a new guid, userful as default value
*   <fh-copy>{{objIdent|JS}}</fh-copy>: In the filter parameter you can use this token wich allows you to autogenerate the object identity
*   <fh-copy>{{Filename.png|file}}</fh-copy>: You can reference previous uploaded resources using this token

## Personalized edit form

*   <fh-copy>{{getProperties(json)}}</fh-copy>: Use to paint object properties on edit form
