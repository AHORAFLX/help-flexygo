# Useful tokens

Use this tokens when you're developing your app to obtain some useful information just by adding them to your code.

*   <fh-copy class="link">{{currentDate}}</fh-copy>: Get current date, useful as default value
*   <fh-copy class="link">{{currentTime}}</fh-copy>: Get current time, useful as default value
*   <fh-copy class="link">{{currentDateTime}}</fh-copy>: Get current date and time, useful as default value
*   <fh-copy class="link">{{currentLocation}}</fh-copy>: Get current user location (altitude, longitude), useful as default value
*   <fh-copy class="link">{{currentAltitude}}</fh-copy>: Get current user altitude, useful as default value
*   <fh-copy class="link">{{currentLongitude}}</fh-copy>: Get current user longitude, useful as default value
*   <fh-copy class="link">{{currentLatitude}}</fh-copy>: Get current user latitude, useful as default value
*   <fh-copy class="link">{{newGuid}}</fh-copy>: Generate a new guid, userful as default value
*   <fh-copy class="link">{{objIdent|JS}}</fh-copy>: In the filter parameter you can use this token wich allows you to autogenerate the object identity
*   <fh-copy class="link">{{Filename.png|file}}</fh-copy>: You can reference previous uploaded resources using this token

## Personalized edit form

*   <fh-copy class="link">{{getProperties(json)}}</fh-copy>: Use to paint object properties on edit form
