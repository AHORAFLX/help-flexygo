# Information

An addon is an additional feature of the tool that can be installed separately and used in different applications.

## Marketplace

From the marketplace located inside flexygo in the administration panel under **Other Tools** -> **Addons** you can install the available addons.

You may need to acquire the corresponding license module from AHORA Business Hub in order to use it.

## How do I develop an addon?

To develop an addon, you can find the necessary information at: [https://docs.flexygo.com/addon.html](https://docs.flexygo.com/addon.html)
