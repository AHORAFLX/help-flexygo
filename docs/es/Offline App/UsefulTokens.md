# Useful tokens

Use this tokens when you're developing your app to obtain some useful information just by adding them to your code.

*   {{currentDate}}: Get current date, useful as default value
*   {{currentTime}}: Get current time, useful as default value
*   {{currentDateTime}}: Get current date and time, useful as default value
*   {{currentLocation}}: Get current user location (altitude, longitude), useful as default value
*   {{currentAltitude}}: Get current user altitude, useful as default value
*   {{currentLongitude}}: Get current user longitude, useful as default value
*   {{currentLatitude}}: Get current user latitude, useful as default value
*   {{newGuid}}: Generate a new guid, userful as default value
*   {{objIdent|JS}}: In the filter parameter you can use this token wich allows you to autogenerate the object identity
*   {{Filename.png|file}}: You can reference previous uploaded resources using this token

##### Personalized edit form

*   {{getProperties(json)}}: Use to paint object properties on edit form
