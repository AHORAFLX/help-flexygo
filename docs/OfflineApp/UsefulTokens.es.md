# Useful tokens

Use this tokens when you're developing your app to obtain some useful information just by adding them to your code.

*    <fh-copy>{{currentDate}}</fh-copy>: Obtiene la fecha actual, útil como valor predeterminado
*    <fh-copy>{{currentTime}}</fh-copy>: Obtiene la hora actual, útil como valor predeterminado
*    <fh-copy>{{currentDateTime}}</fh-copy>: Obtiene la fecha y hora actuales, útil como valor predeterminado
*    <fh-copy>{{currentLocation}}</fh-copy>: Obtiene la ubicación actual del usuario (altitud, longitud), útil como valor predeterminado
*    <fh-copy>{{currentAltitude}}</fh-copy>: Obtiene la altitud actual del usuario, útil como valor predeterminado
*    <fh-copy>{{currentLongitude}}</fh-copy>: Obtiene la longitud actual del usuario, útil como valor predeterminado
*    <fh-copy>{{currentLatitude}}</fh-copy>: Obtiene la latitud actual del usuario, útil como valor predeterminado
*    <fh-copy>{{newGuid}}</fh-copy>: Genera un nuevo GUID, útil como valor predeterminado
*    <fh-copy>{{objIdent|JS}}</fh-copy>: En el parámetro de filtro puedes usar este token que te permite autogenerar la identidad del objeto
*    <fh-copy>{{Filename.png|file}}</fh-copy>: Puedes referenciar recursos previamente cargados usando este token


## Formulario de edición personalizado

*   <fh-copy>{{getProperties(json)}}</fh-copy>: Utiliza para pintar las propiedades del objeto en el formulario de edición
